# Calculadora Simples

Uma calculadora básica em Python que realiza operações matemáticas simples, como adição, subtração, multiplicação e divisão. Esta calculadora possui uma interface gráfica construída usando Tkinter.

## Descrição

Este projeto é uma calculadora de interface gráfica simples feita com Python e Tkinter. A calculadora permite realizar operações matemáticas básicas:

- Adição (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)

## Tecnologias Utilizadas

- **Python**: Linguagem de programação utilizada para desenvolver a calculadora.
- **Tkinter**: Biblioteca padrão do Python para criação de interfaces gráficas.

## Como Usar

1. **Clone o repositório**:

   ```bash
   git clone https://github.com/seu-usuario/calculadora-simples.git
   ```

2. **Navegue até o diretório do projeto**:

   ```bash
   cd calculadoraTK
   ```

3. **Execute o script da calculadora**:

   ```bash
   python calculadoraTK.py
   ```

   **Nota:** Certifique-se de que você tem o Python instalado na sua máquina. Tkinter é uma biblioteca padrão e já vem com o Python, então não é necessário instalar separadamente.

## Funcionalidades

- **Interface Gráfica**: Interface de usuário simples e fácil de usar.
- **Operações Básicas**: Realiza operações matemáticas básicas.
- **Limpeza e Cálculo**: Botões para limpar o visor e calcular resultados.
